# Flutter Funcionários

## Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo Flutter chamado **flutterfuncionarios**, responsável por exibir uma lista de funcionários a partir de um arquivo JSON local.

Os dados são carregados do arquivo:
`assets/mockup/funcionarios.json`

O campo **avatar** deve ser adicionado manualmente ao JSON, utilizando links de imagens da web.

O aplicativo possui duas telas principais:
- Uma tela Splash com animação (movimento, escala ou opacidade)  
- Uma tela Home que exibe os funcionários e permite navegar entre eles  

---

## Tecnologias Utilizadas

- Flutter  
- Dart  
- Material Design  

---

##  Como Executar o Projeto

1. Clone o repositório:
```bash
git clone 
```

2. Acesse a pasta do projeto:
```bash
cd flutter_funcionarios
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Execute o aplicativo:
```bash
flutter run
```

---

##  Funcionamento do Aplicativo

1. Ao iniciar, o aplicativo exibe uma tela Splash com animação  
2. Em seguida, o usuário é direcionado para a tela principal (Home)  
3. O sistema carrega os dados do arquivo JSON  
4. Um funcionário é exibido por vez  

Cada funcionário exibe:
- Nome  
- Cargo   
- Imagem (avatar)
